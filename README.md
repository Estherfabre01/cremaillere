# Invitation crémaillère — GitHub Pages

## Fichiers
- `index.html` : page responsive complète.
- `images/jardin.jpg` : image à ajouter/remplacer par ta photo.

## À personnaliser
1. Remplace `images/jardin.jpg` par une photo de ton jardin ou de la maison.
2. Remplace `FORM_ACTION_ICI` dans `index.html` par :
   - ton lien Google Forms, ou
   - une URL Formspree si tu veux collecter les réponses directement depuis le formulaire HTML.
3. Publie le dépôt avec GitHub Pages.
